# SmartShield – Real-Time URL Phishing Detection

SmartShield is a browser-based phishing detection system that analyzes URLs in real time before page rendering.

## Features
- Real-time URL analysis
- Machine Learning based detection
- Flask backend API
- Browser extension integration
- TF-IDF character-level feature extraction
- Random Forest classification
- Feedback collection for future retraining

## Tech Stack
- Python
- Flask
- Scikit-learn
- NumPy
- Browser Extension

## Run Locally

Install dependencies:

pip install -r requirements.txt

Start backend:

python backend/app.py

Open browser extension and connect to backend.
